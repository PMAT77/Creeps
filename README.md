# Creeps

## 📝 项目介绍

项目基于 Vite + TypeScript 构建，包含的模块有：

- GameController 核心类
- GameState 游戏状态类
- Entity 实体类
  - Creep 玩家和敌对单位类
  - Bullet 子弹基类

## ✅ 已实现功能

- `creep` 单位移动转向: WSAD
- `creep` 单位单独转向: ⬆️ ⬇️ ⬅️ ➡️
- `creep` 单位发射子弹: `自动射击` 或者 `方向键射击`

## ❓ 如何运行项目

**安装依赖**

```
npm install
```

**运行项目**

```
npm run dev
```

**打包项目**

```
npm run build
```
